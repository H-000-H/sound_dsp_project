# EQ 滤波器原理

## 核心参数

### 品质因素 Q
```
Q = f₀ / BW
```

### Peak Filter 带宽计算
峰值滤波器的上下限频率按倍频关系设计：
```
f₁ = f₀ / 2^(n/2)
f₂ = f₀ × 2^(n/2)
带宽 Δf = f₂ - f₁ = (2^(n/2) - 1/2^(n/2)) × f₀
品质因素 Q = f₀ / Δf = 2^(n/2) / (2^n - 1)
```

### 增益（Gain）
中心频率处的增益（dB）：
```
gain(dB) = 20 × lg(H(f₀) / H(f₀'))
```

## 二阶 IIR 滤波器基础

### 传递函数
```
H(z) = (b₀ + b₁z⁻¹ + b₂z⁻²) / (a₀ + a₁z⁻¹ + a₂z⁻²)
```

### 时域差分方程
```
y(n) = [b₀·x(n) + b₁·x(n-1) + b₂·x(n-2) - a₁·y(n-1) - a₂·y(n-2)] / a₀
```

### 关键中间变量（通用）
```
ω₀ = 2π × f₀ / f_s    （数字角频率）
α = sin(ω₀) / (2 × Q)  （结合Q值的步长系数）
A = 10^(dB_Gain / 40)  （增益放大因子，仅限增益型滤波器）
```

## 滤波器系数计算公式

### 1. Peaking EQ（尖峰/钟形均衡器）
```
b₀ = 1 + α × A
b₁ = -2 × cos(ω₀)
b₂ = 1 - α × A
a₀ = 1 + α / A
a₁ = -2 × cos(ω₀)
a₂ = 1 - α / A
```

### 2. Low Shelf EQ（低架均衡器）
```
b₀ = A × [(A+1) - (A-1)×cos(ω₀) + 2×√A×α]
b₁ = 2×A × [(A-1) - (A+1)×cos(ω₀)]
b₂ = A × [(A+1) - (A-1)×cos(ω₀) - 2×√A×α]
a₀ = (A+1) + (A-1)×cos(ω₀) + 2×√A×α
a₁ = -2 × [(A-1) + (A+1)×cos(ω₀)]
a₂ = (A+1) + (A-1)×cos(ω₀) - 2×√A×α
```

### 3. High Shelf EQ（高架均衡器）
```
b₀ = A × [(A+1) + (A-1)×cos(ω₀) + 2×√A×α]
b₁ = -2×A × [(A-1) + (A+1)×cos(ω₀)]
b₂ = A × [(A+1) + (A-1)×cos(ω₀) - 2×√A×α]
a₀ = (A+1) - (A-1)×cos(ω₀) + 2×√A×α
a₁ = 2 × [(A-1) - (A+1)×cos(ω₀)]
a₂ = (A+1) - (A-1)×cos(ω₀) - 2×√A×α
```

### 4. Low Pass Filter（低通滤波器 / 高切）
```
b₀ = (1 - cos(ω₀)) / 2
b₁ = 1 - cos(ω₀)
b₂ = (1 - cos(ω₀)) / 2
a₀ = 1 + α
a₁ = -2 × cos(ω₀)
a₂ = 1 - α
```

### 5. High Pass Filter（高通滤波器 / 低切）
```
b₀ = (1 + cos(ω₀)) / 2
b₁ = -(1 + cos(ω₀))
b₂ = (1 + cos(ω₀)) / 2
a₀ = 1 + α
a₁ = -2 × cos(ω₀)
a₂ = 1 - α
```

### 6. Band Pass Filter（带通滤波器）
```
b₀ = α
b₁ = 0
b₂ = -α
a₀ = 1 + α
a₁ = -2 × cos(ω₀)
a₂ = 1 - α
```

## 数字角频率 ω₀ 说明

### 频率映射关系
| 模拟频率 f | 数字角频率 ω |
|-----------|-------------|
| 0 Hz | 0 rad |
| f_s/4 | π/2 rad |
| f_s/2（奈奎斯特频率） | π rad |

### 核心特性
- **归一化**：ω 是无量纲的，与实际采样率无关
- **周期性**：ω + 2π 等价于 ω
- **Z平面意义**：极点位置 z = e^(jω) = cos(ω) + j·sin(ω)
