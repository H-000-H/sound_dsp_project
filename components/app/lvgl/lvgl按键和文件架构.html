#按键映射

按键GPIO锁屏卡片菜单应用内
PREVCONFIG_LVGL_KEY_PREV_GPIO—左滑卡片焦点上移 / 减小值
NEXTCONFIG_LVGL_KEY_NEXT_GPIO—右滑卡片焦点下移 / 增大值
ENTERLV_EVENT_KEY解锁 → 卡片菜单进入应用进入调整模式 / 触发动作
ESCLV_EVENT_KEY—回锁屏退出调整 / 返回上级

按键检测流程：GPIO 中断 → ISR 入队 FreeRTOS 队列 → 主循环 Button::process() 出队 → get_pressed_gpio() 读取当前键值
ENTER/ESC 通过 LV_EVENT_KEY 事件 + lv_indev_t 映射到 KEY 设备 key_board_cb 回调

#App 焦点环

每个 App 内部用 PREV/NEXT 在焦点项间循环，ENTER 进入调整模式，再次 ENTER 确认退出，ESC 退出调整。

⚙️Settings · 设置

焦点环顺序

  • WiFi 开关
  • 蓝牙开关
  • MQTT 开关
  • 亮度调节
  • 低功耗开关

调整模式行为

  • WiFi/蓝牙/MQTT/低功耗：ENTER 切换开关
  • 亮度：ENTER +5，ESC -5
  • WiFi/BT/MQTT 操作通过 lvgl_defer 异步到 Core 1 执行

🎵Music · 音乐播放器

焦点环顺序

  • 选歌列表
  • 播放进度条
  • 播放模式（顺序/随机/单曲循环）
  • 播放控制（上一曲/播放暂停/下一曲）
  • 音效模式（无/预设 EQ）

调整模式行为

  • 进度/模式/音效：PREV/NEXT 调值，ENTER 确认
  • 播放控制：ENTER 播放/暂停，PREV 上一曲，NEXT 下一曲
  • 选歌：ENTER 进入 SongListPage 子页面
  • 子页面 5 首/页，圆点分页，ENTER 选歌跳回播放页

🔌Serial · 串口终端

焦点环顺序

  • 波特率选择
  • 终端模式（HEX / 文本）
  • 滚动控制

调整模式行为

  • ENTER 循环切换值
  • 波特率：115200 / 57600 / 38400 / 19200 / 9600
  • 终端模式切换 HEX 与文本显示
  • 滚动锁定/解锁终端输出

#关键数据流

场景路径说明
WiFi WiFi 开关 SettingsApp::do_enter → lvgl_defer → SettingsImpl::on_wifi_toggle defer 到 LVGL 任务外执行，避免死锁
音乐 歌曲播放 MusicImpl::on_song_select → DecodeTask → I2S 输出 MusicImpl 通过任务通知触发解码
音乐 进度计时 progress_timer_cb（1 秒）→ m_elapsed++ → update_display 静态成员函数，每秒更新进度条和时间
音乐 选歌跳转 SongListPage ENTER → MusicImpl::on_song_select → 返回播放器 SongListPage 是 MusicImpl 的友元
系统 锁屏计时 lock_screen 定时器（1 秒）→ factory_config::time::get_time() 每秒刷新锁屏时间显示
导航 按键检测 GPIO 中断 → FreeRTOS 队列 → Button::process() → key_board_cb 中断级入队，任务级出队，LVGL 回调消费
