按键映射

按键GPIO锁屏卡片菜单应用内
PREVCONFIG_LVGL_KEY_PREV_GPIO—左滑卡片焦点上移 / 减小值
NEXTCONFIG_LVGL_KEY_NEXT_GPIO—右滑卡片焦点下移 / 增大值
ENTERLV_EVENT_KEY解锁 → 卡片菜单进入应用进入调整模式 / 触发动作
ESCLV_EVENT_KEY—回锁屏退出调整 / 返回上级

按键检测流程：GPIO 中断 → ISR 入队 FreeRTOS 队列 → 主循环 Button::process() 出队 → get_pressed_gpio() 读取当前键值
ENTER/ESC 通过 LV_EVENT_KEY 事件 + lv_indev_t 映射到 KEY 设备 key_board_cb 回调

App 焦点环

每个 App 内部用 PREV/NEXT 在焦点项间循环，ENTER 进入调整模式，再次 ENTER 确认退出，ESC 退出调整。

App焦点环顺序调整模式行为
设置 WiFi → 蓝牙 → MQTT → 亮度 → 低功耗 WiFi/蓝牙/MQTT/低功耗：ENTER 切换开关
亮度：ENTER +5，ESC -5
音乐 选歌 → 进度 → 播放模式 → 播放控制 → 音效 进度/模式/音效：PREV/NEXT 调值，ENTER 确认
播放控制：ENTER 播放/暂停，PREV/NEXT 切歌
串口 波特率 → 终端模式 → 滚动控制 ENTER 循环切值

关键数据流

场景路径说明
WiFi 开关 SettingsApp::do_enter → lvgl_defer → SettingsImpl::on_wifi_toggle defer 到 LVGL 任务外执行，避免死锁
歌曲播放 MusicImpl::on_song_select → DecodeTask → I2S 输出 MusicImpl 通过任务通知触发解码
进度计时 progress_timer_cb（1 秒）→ m_elapsed++ → update_display 静态成员函数，每秒更新进度条和时间
选歌跳转 SongListPage ENTER → MusicImpl::on_song_select → 返回播放器 SongListPage 是 MusicImpl 的友元
锁屏计时 lock_screen 定时器（1 秒）→ factory_config::time::get_time 每秒刷新锁屏时间显示
